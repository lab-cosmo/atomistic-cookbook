Path integral molecular dynamics of bosons and fermions
========================================================

This example shows how to run path integral molecular dynamics simulations of
indistinguishable particles -- bosons and fermions -- with ``i-PI``, using the
built-in harmonic Python driver, and how to reweight bosonic runs to obtain
fermionic averages via the native ``fermionic_sign`` property. It modernises the
"Bosonic and Fermionic PIMD" module of the PIQM 2023 tutorial for i-PI 3.x.
