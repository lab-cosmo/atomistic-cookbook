NMR-driven structure determination with ShiftML3
================================================

This recipe shows how to use the
`ShiftML3 <https://github.com/lab-cosmo/shiftml>`_ machine-learning model
of NMR chemical shieldings to identify the experimental crystal structure
of a molecular solid from a pool of candidate polymorphs, by comparing
predicted :sup:`1`\ H chemical shifts to a measured solid-state NMR
spectrum.
