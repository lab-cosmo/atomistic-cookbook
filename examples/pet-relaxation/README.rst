Geometry relaxation with unconstrained MLIPs
=============================================

This recipe demonstrates how unconstrained machine-learning potentials
like PET-MAD exploit residual symmetry-breaking noise to escape saddle
points during geometry optimization.

Two systems are explored:

1. **Al (Bain path):** BCC → FCC via spontaneous cell shear.
2. **BaTiO₃ (perovskite):** unconstrained relaxation discovers the
   ferroelectric :math:`R3m` phase from the cubic structure.
